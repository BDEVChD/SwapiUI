## Follow these steps to install and setup a local UI

### Step 1: 

Use composer to install all dependencies by typing the following command: 

composer install 

```

### Step 2: 

Create the `.env` file
$ cp .env.example .env


```

### Step 3: 

Generate an app key
$ php artisan key:generate

### Step 4: 

Run a local artisan server

```
php artisan serve

```
